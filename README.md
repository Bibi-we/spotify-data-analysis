Spotify Data Analysis

Overview

This project explores Spotify track data using Python, Pandas, Matplotlib, Seaborn, and SQLite.

The goal is to analyze song popularity, artist performance, audio features, and relationships between track characteristics.

Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- SQLite
- Git
- GitHub

Analysis Performed

Data Exploration

- Dataset inspection
- Missing value checks
- Summary statistics

Visualizations

- Top 10 Songs by Popularity
- Popularity Distribution
- Popularity vs Danceability
- Popularity vs Energy
- Correlation Heatmap
- Top Artists by Average Popularity
- Top Artists by Number of Songs
- Top Artists by Average Energy

Correlation Analysis

Features analyzed:

- Popularity
- Danceability
- Energy
- Tempo

SQL Analysis

Implemented SQLite queries for:

- Top 10 songs by popularity
- Top artists by average popularity
- Artists with the most songs
- Artists with highest average energy

Key Findings

- Energy shows a stronger relationship with popularity than danceability.
- Tempo has a weak relationship with popularity.
- Several artists have large catalogs but relatively low average popularity.
- SQL and Pandas produce equivalent analytical results through different approaches.
